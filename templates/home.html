Welcome to Outernet Content Selection and Discovery System (CSDS). This site is part of Outernet project, and a testbed for the production CSDS. Feel free to kick the tires and let us know what you think.

For those familar with the project, please keep in mind that Outernet is not live yet. So, everything here is just a simulation. However, data gathered during test runs _will_ influence the final systems, so the role of test users is quite important, possibly more so now than later. We thank you in advance for taking part in developing Outernet.

How? What? Where?

Currently, the only means of submitting requests is via the Outernet's Facebook page. Anything posted on this page is treated as a request for content and will be harvested every 30 minutes by our server. The harvested requests are put in a queue, and await for content suggestions.

You can see the requests that are in the system by going to the request list page.

You can also find a list of community-suggested content URLs that belong to unbroadcast requests. This list is known as 'content pool' and it is used to build daily playlists. The pool is sorted by total number of votes but the sorting order is subject to change in future releases.

For each harvested request, you can make content suggestions. This is currently quite simple: just paste the page's address (URL) and hit "Suggest". The URL must be valid (should not be missing or return errors), and duplicate URLs are not accepted.

You can also vote on suggested content.

Each day, a playlist is built from the suggested content. This will eventually be done completely automatically with occasional manual intervention from volunteers. The playlist items are selected from the content pool, and this is currently exposed as a completely manual UI. The current state of the today's playlist can be seen on the playlist page.

No user accoutns

Four parts laziness and six parts geniuine wish for a completely anonymous system, the concept version of CSDS has no user accounts. This gives you freedom to (ab)use the system in any way conceivable. We hope to learn from this as much as we can before we start imposing any form of constraint.
